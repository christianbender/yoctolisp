# Lisp interpreter

Yoctolisp is a Lisp interpreter written in a weekend.  

---

## Features  

 - Read-eval-print loop.  
 - Lexical closures.  
 - Proper garbage collection.  
 - Tail call optimization.  
 - Variadic function syntax.
